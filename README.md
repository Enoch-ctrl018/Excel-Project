# Excel Public Hospital Project

## Project Overview
This project analyzes public hospital utilization data using Microsoft Excel. The project focuses on organizing, cleaning, and analyzing healthcare data to identify trends and generate meaningful insights through charts, tables, and Excel functions.

## Tools Used
- Microsoft Excel

## Project Tasks
- Data cleaning and organization
- Sorting and filtering records
- Creating charts and tables
- Trend analysis
- Data interpretation and reporting

## Key Analysis Performed
- Analyzed hospital utilization patterns
- Identified trends within the dataset
- Created visual representations using charts and tables
- Organized raw data for easier interpretation

## Files Included
- Excel workbook (.xlsx)
- Project screenshots

## Conclusion
This project demonstrates the use of Microsoft Excel for data cleaning, analysis, visualization, and reporting.
